Getting Started

If you haven't already, read the Terms of Service.

First, add SwagBot to your server. You don't need to give SwagBot the Administrator permission, but some features may not work. The bot won't do anything besides what you tell it! SwagBot requires the Embed Links permission as almost all commands use embeds to format messages.

Now test if the bot is functioning properly by typing ~ping. You should get a response that says Pong!. If not, make sure the bot has permission to send messages.

From here you can type ~help to see a list of commands, or you can go here for a more descriptive list. You can always use ~help [command] to get specific information about a specific command.

Some commands require permissions within the bot. To learn more about this, go to the permissions page.
